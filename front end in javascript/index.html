EXECUTING THE PROJECT

The CRUD REST API is built in PHP using Laravel Framework. The front-end is written in JavaScript using Material Design For BootStrap Library!

  • To Run the php server , Go to the root directory and type php artisan serve
  • Mysql configuration is required to use the database.
  • .Env file needs to be configured for the database which could be found in the root of the php project.
  • First database needs to be created.DB_DATABASE = mydb, DB_NAME=mydb,DB_USERNAME=root,DB_PASSWORD=
  • After setting up the database, database migration needs to done by php artisan migrate.
